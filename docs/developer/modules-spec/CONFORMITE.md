# Conformité

## Rôle

Définir et vérifier les exigences de conformité applicables aux personnes,
documents, exemptions et règles.

## Fonctions

- types de documents
- règles de conformité
- vérifications
- exemptions
- tableaux de bord conformité

## Dépendances

- PaxLog pour les PAX et les séjours
- Asset Registry pour le contexte site
- Core pour permissions, audit, notifications
