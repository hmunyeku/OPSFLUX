# Pack de travail - Evaluation OVA OpsFlux

Ce dossier regroupe les documents de travail pour organiser l'evaluation OpsFlux avec OVA.

L'objectif n'est pas de faire de ces premiers lots la finalite du projet. Ces lots servent de socle pour tester la plateforme, remonter les anomalies, verifier la documentation utilisateur et peupler les donnees avant d'aborder les modules metiers.

## Classeur Excel de pilotage

Le tableau de suivi operationnel est disponible au format Excel:

- [tableau-suivi-tests-ova.xlsx](./tableau-suivi-tests-ova.xlsx)

Il contient une granularite fine de recette avec 205 points de controle pre-remplis: acces, profils, comptes, droits, assets, conformite, support, tiers, projets, securite, responsive, traductions, documentation et donnees.

## Utilisation proposee

- OVA execute les scenarios du tableau de suivi.
- Chaque anomalie est documentee avec le modele de fiche.
- Les donnees fournies sont tracees dans le catalogue.
- Les droits et comptes de test sont controles avec la matrice.
- Les retours servent a stabiliser le socle avant les modules metiers.
